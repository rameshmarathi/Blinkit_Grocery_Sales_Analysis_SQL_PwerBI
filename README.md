# Blinkit Sales Analysis using SQL and Power BI

![Sales Analysis Dashboard](pivot-sales-analysis.png)

*This is my dashboard image.*

## Project Overview

This project focuses on analyzing Blinkit sales data using SQL and Power BI.
The main objective of this project is to clean the data, perform KPI analysis, and create interactive dashboards for business insights.

---

## Tools & Technologies Used

* SQL
* Power BI
* Excel / CSV Dataset
* GitHub

---

## Key Features

### Data Cleaning

* Standardized inconsistent values in `Item_Fat_Content`
* Removed data inconsistencies using SQL UPDATE queries

### KPI Analysis

* Total Sales
* Average Sales
* Number of Items
* Average Rating

### Business Insights

* Sales by Fat Content
* Sales by Item Type
* Sales by Outlet Size
* Sales by Outlet Location
* Outlet Performance Analysis

### Advanced SQL Concepts

* GROUP BY
* ORDER BY
* Aggregate Functions
* PIVOT Queries
* Window Functions
